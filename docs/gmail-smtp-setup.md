# Gmail App Password Setup

Follow these steps to set up a Gmail App Password for the O.N.E. Onboarding Agent:

1. Go to `myaccount.google.com`
2. Security -> 2-Step Verification -> enable it
3. Security -> App passwords -> create one called "ONE Onboarding Agent"
4. Copy the 16-character app password
5. Set `GMAIL_ADDRESS` and `GMAIL_APP_PASSWORD` in your `.env` file.
